  This sample demonstrates how web pages can request images
  from a websocket server (pictureserver.exe), and how those
  images can be displayed on a web page.

Attention:

  Although the images are correctly delivered to Opera web
  browsers, a bug in Opera prevents the images from being 
  displayed correctly after the transfer.
