The paragraph tag <p>

A paragraph begins with the HTML tag <p> and ends with the corresponding </p> tag. Text enclosed by these tags will be wrapped when the window with is reduced, even if it contains line break characters (CR + LF).

A line break is indicated by the tag <br> in the HTML format.

Alignment of paragraphs

Attributes can be defined for the <p> Tag which define how a paragraph is aligned. The attributes are: <p align=left|center|right>.
