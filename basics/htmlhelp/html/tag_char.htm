Font attributes

The default font attributes Bold, Underlined and Italic begin with the HTML tags <b>, <u> und <i> and must end with the corresponding tags </b>, </u> und </i>.

Fonts

Fonts are selected in HTML with the <font> tag. The attributes face=, and size= are used to select size and type of a font. The font size is defined using values from 1 through 7 (default size is 3). Whether or not a font can be displayed depends on the HTML browser and on the installed fonts.

Special characters

Special characters can be defined using special formatting. It begins with the character &, followed by the name of the character and a semicolon:

Ä = &Auml;
ä = &auml;
ß = &szlig;
® = &reg;
© = &copy;
< = &lt;
> = &gt;
