There are 6 HTML-Tags which are used for formatting headings.

<h1> - Formats headings of the first level

<h2> - Formats headings of the second level

<h3> - Formats headings of the third level

<h4> - Formats headings of the fourth level

<h5> - Formats headings of the fifth level

<h6> - Formats headings of the sixth level

Each <h> tag must end with the correspondin </h> tag.
